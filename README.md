messages
========

## Install

Install with yarn.

```sh
$ yarn
```

Otherwise, install with npm.

```sh
$ npm install
```

## Build

```sh
$ npm run build
```

## Start

```sh
$ npm start
```

## Docs

```sh
$ npm run docs
```

## Database

### Update database

```sh
$ npm run latest
```

### Rollback database update

```sh
$ npm run rollback
```

### Create new migration

```sh
$ npm run make-migration
```

Creates a new migration file in `db/migrations`.
